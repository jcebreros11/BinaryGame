# Binary Game

A fast-paced cyber-themed React game where players convert binary numbers into base-10 before the timer runs out.

## Live Demo

🎮 https://binary-game-hqdd.vercel.app/

---

## Features

- Multiple difficulty modes
  - Easy
  - Medium
  - Hard
  - Custom settings

- Custom game configuration
  - Adjustable timer
  - Adjustable min/max range
  - Max value capped at 255

- Real-time countdown timer

- Score tracking

- Streak system 🔥

- Responsive mobile-friendly UI

- Smooth transitions and animated loaders

---

## Tech Stack

- React
- Vite
- React Router
- CSS3

---

## How To Play

1. Select a difficulty
2. Convert the binary number shown on screen into base-10
3. Submit your answer before the timer expires
4. Build your streak and beat your high score

Example:

```txt
Binary: 1010
Answer: 10
